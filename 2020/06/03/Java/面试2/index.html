Java面试题数据库篇

Jun 03, 2020Java

数据库的分类

关系型数据库和非关系型数据库

  • 关系型数据库：Mysql，Oracle，SQL Server
  • 非关系型数据库：Redis，memcache，mogodb，hadoop

关系型数据库三范式

  • 第一范式（1NF）：字段不可再分，一个字段只存储一项信息，如果实体的某个属性有多个值，必须拆分成不同的属性。
  • 第二范式（2NF）：有主键，非主键字段依赖主键，任意一个字段都只依赖表中的同一个字段
  • 第三范式（3NF）：非主键字段不能相互依赖，数据不能存在传递关系（非主键字段之间没有依赖传递）

事务的四大特征(ACID)

  • 原子性：事务内操作不可分割，要么都成功，要么都失败
  • 一致性：事务前后数据的完整性要保持一致，如果事务失败要进行回滚操作
  • 隔离性：多个并发事务之间相互隔离，互不影响
  • 持久性：事务对数据的改变是永久性的

mysql数据库的默认最大连接数

100

触发器

触发器用来保证数据完整性，由时间来触发，比如当对一个表进行插入操作是就会激活它，使用场景是当操作某张表时，希望同时触发一些动作，就可以使用触发器。（触发器是一种特殊的存储过程）

存储过程

存储过程是一组为了完成特定功能的SQL语句，这些SQL语句代码像一个方法一样实现一些功能

  • 只在创造的时候进行编译，以后的每次执行都不需要再编译，可以提高数据库执行速度
  • 复杂的业务逻辑通常需要多条SQL语句，在网络中交互时，存储过程能够降低网络的通信量，提高通信速率
  • 可以重复使用，减少数据库开发人员的工作量
  • 安全性高，可设定只有某些用户才具有对指定存储过程的使用权
  • 没有版本控制

如果SQL复杂，或者数据交互多且中间数据最后不会用到，使用存储过程

PreparedStatement相比Statement的好处

PreparedStatement预编译，比Statement速度快

PreparedStaement可以防止SQL注入攻击

数据库连接池的作用

  • 可以避免频繁创建、释放连接引起的大量性能开销
  • 在数据库连接池初始化的过程中，已经创建了若干数据库连接用于备用，所以业务请求可以利用现有连接，避免数据路连接初始化和释放的时间开销，缩短系统响应时间
  • 统一的连接管理，避免数据路连接泄露
